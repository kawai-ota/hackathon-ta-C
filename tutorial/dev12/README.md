tutorial-nodejs
====

「実際に動かしてみましょう」のソースコード

## アプリの起動

（例）`developer1`ユーザが`sample1`のサンプルアプリを起動する場合

1. 次のコマンドを実行します

    ```bash
    # アプリのディレクトリに移動
    cd ~/hackathon/tutorial/developer1/sample1
    # アプリの開発で使用するライブラリをインストール（初回のみ）
    npm install
    # 起動コマンドを実行
    npm start
    ```

2. `http://サーバのIPアドレス:ポート番号/` にブラウザでアクセスします

    例：`http://127.0.0.1:3000/`
